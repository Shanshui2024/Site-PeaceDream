# PeaceDream - 和梦团队网站

和梦团队官方网站，提供 Minecraft 整合包下载和剪辑资源分享。

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM

## 功能特性

- 响应式设计，支持移动端和桌面端
- 简洁干净的页面设计
- 共用组件（导航栏、页脚）
- 分页面设计（主页、下载页）

## 页面结构

### 主页 (/)
- 团队介绍
- SakuraFRP 状态信息
- 快速链接到下载页和 QQ 群

### 下载页 (/download)
- 下载必看（注意事项）
- MC整合包分享（11 个整合包）
- 剪辑资源分享（3 个资源）

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
